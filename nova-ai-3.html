🎙 Sesli Komutlar

    

      
"Nova" → Sesli asistanı aç

      
"Nova, sabah 7'de uyandır" → Alarm kur

      
"Nova, WhatsApp'ta mesaj yaz" → WhatsApp aç

      
"Nova, müzik çal" → Müzik uygulaması

      
"Nova, harita aç" → Harita uygulaması

      
"Nova, not al: ..." → Görev listesine ekle

      
"Nova, fotoğraf çek" → Kamerayı aç

      
"Nova, [herhangi soru]" → AI'a sor

    

    

      ⚠️ HTTPS veya localhost gerektirir. Mikrofon izni istenir.
    

  







  

    ❓ Yardım & Kılavuz
    
  

  

    
    
🚀 HIZLI BAŞLANGIÇ

    

      
1️⃣ API Key → 🔑 butonuna bas, anahtarını gir

      
2️⃣ Model seç → Üstteki ⚡🤖 ikonlardan seç

      
3️⃣ Mesaj yaz → Alt kutuya yaz, Enter ile gönder

      
4️⃣ FAB menü → Sağ alttaki ✦ — 100+ özellik

    

    
    
⌨️ KLAVYE KISAYOLLARI

    

      

        EnterMesaj gönder
        Shift+EnterSatır sonu
        Ctrl+KSohbet ara
        Alt+VSesli asistan
        EscPanel kapat
      

    

    
    
🌙 DARK MOD

    

      Mesaj kutusuna @1#@1# yaz → Dark mode aktif

"Nova" dersen uyanır

      
"Nova, sabah 7'de uyandır" → Alarm

      
"Nova, WhatsApp'ta mesaj yaz" → Uygulama
